# atom-x

Lightweight, zero-dependency TypeScript date utilities.

Designed for excellent package discoverability and distribution:

- npm metadata optimized (description, keywords, repository, homepage, bugs)
- CDN ready via unpkg/jsDelivr/UMD bundle
- Search-friendly keywords for SEO/GEO use cases

## Install

```bash
npm i atom-x
```

## Usage

```ts
import { format, fromNow, getDuration } from "atom-x/date";
import { format as formatDate } from "atom-x";
```

## Scripts

- `npm run build`
- `npm run lint`
- `npm run test`
